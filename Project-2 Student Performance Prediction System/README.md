# 🎓 Student Performance Prediction System

A menu-driven Machine Learning project that predicts a student's final performance based on various academic factors such as attendance, study hours, internal marks, assignment scores, and previous marks.

This project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis, visualization, feature selection, model training, and prediction.

---

## 📌 Project Objective

The objective of this project is to predict a student's final academic performance using Machine Learning techniques.

The system allows users to:

- Generate a dataset of 200 students
- Analyze student data
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Visualize data using charts
- Train a Machine Learning model
- Evaluate model accuracy
- Predict student performance
